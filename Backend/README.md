# Backend

## ITM Buildserver Setup

Om de ITM Buildserver te maken, volg je deze stappen:

1. **Installeer git:**
    ```bash
    sudo apt install git
    ```
    
2. **Kloon Repository:**
    ```bash
    git clone https://github.com/GuidoAsbroek/FTS.git
    ```

3. **Draai het Script:**
    ```bash
    cd FTS/Backend
    ./setup.sh
    ```

3. **Test Setup:**
    De Backend van Fleet Tracker draait op http://localhost/api:80
